## Setup
#### Project Properties
##### VC++ Directories (Library Directories, Include Directories)

`<Your sdk folder>\Include;$(IncludePath)`

`<Your sdk folder>\Lib\x86;$(LibraryPath)`

##### Linker Input
`d3dx9.lib;d3d9.lib;dxguid.lib;dxerr.lib;dinput8.lib;%(AdditionalDependencies)`
